IntelliJ IDEA

安装

jetbrains下载IntelliJ IDEA Community Edition2021版

https://www.jetbrains.com/idea/download/other.html

启动

导入Java项目，步骤：

启动启动IntelliJ IDEA->选择open打开项目文件->ide右下角提示“Maven build scripts found”->点击“load Maven project”,IntelliJ IDEA 开始同步 Maven 项目->同步后出现“reload Maven Projects”，选择Recreate。

  • load Maven project：这表示 IntelliJ IDEA 已经检测到项目中存在 Maven 的 pom.xml 文件，这是 Maven 项目的核心配置文件。点击 “Load Maven Project” 来让 IntelliJ IDEA 加载 Maven 项目并处理其依赖关系。

  • reload Maven Projects选择Recreate：尝试重新创建模块并修复任何类型不匹配的问题，确保项目在 IntelliJ IDEA 中正确配置，并且所有 Maven 依赖都能被正确加载。

    按提示，alwaysload一些插件，jdk加载。多次重启后，可以追踪代码。

项目入口文件：入口文件是指包含 main 方法的文件

插件安装：比如，中英文转换

  • 插件安装Chinese (Simplified) Language Pack，重启使插件生效。

  • 取消中文环境：IntelliJ IDEA-》plugin>取消该插件的勾选-》重启恢复语言环境.

代码最新修改人：左侧行数处（空白处），选择annotate，显示最新修改人。

debug配置

配置好数据库后，配置debug参数。

Run > Edit Configurations->Application，添加参数，空格划分参数

调试好，点击 Apply 或 OK 保存配置。选择 Debug 按钮（一个虫子图标）来启动调试会话，通过点击行号左边的空白区域来设置断点。
