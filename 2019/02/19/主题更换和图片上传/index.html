Hexo主题更换和图片上传

一、Hexo博客更换主题：

1、登录https://hexo.io/themes/选择想要的主题

点击图片可以预览所有者的博客

点击图片下方的文字可以进入主题对应的github代码

2、克隆（复制）代码

在项目文件夹下点开git bash

hexo clone git@github.com:wizardforcel/hexo-theme-landfarz.git

克隆得到一个文件夹名为hexo-theme-landfarz，将其剪切放入theme文件夹中

修改_config.yml中的theme： hexo-theme-landfarz（文件夹什么名，就改什么名，注意空格）

3、清空public

hexo clean

4、生成

hexo g（hexo generate生成 静态文件）

5、启动服务器

hexo s

6、打开浏览器，查看本地主题是否更改

localhost:4000(localhos+t你本地仓库服务的端口号)

7、改变后，上传远程仓库

hexo d

8、输入域名https://abellmz.github.io/完成

二、上传图片

1、在_config.yml文件 post_asset_folder 字段设置为true

2、设置 后，建立文件时Hexo 会生成一个与文章同名的文件夹，可以把与本文章相关的所有图片等资源都放到此文件夹内，更方便的调用资源。

3、到博客的根目录下执行 npm install https://github.com/CodeFalling/hexo-asset-image –save 命令来进行插件的安装。

4、执行 hexo new “XXX” 之后查看博客的 ../source/_posts 目录下的文件，会看到存在一个XXX文件夹 和 XXX.md 文件。

5、将所需要的图片资源放到XXX文件夹下,写文章就使用XXX文件内的图片

7、hexo s 命令预览一下博客效果，最后执行hexo clean-> hexo g -> hexo d完成！
