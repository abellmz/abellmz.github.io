解决hexo绑定域名丢失问题

​ 前两天操作hexo 发现，原来绑定在对应GitHub 项目上的域名没了，于是在github上添加上，再次操作 hexo clean、hexo g、hexo d ，结果域名又没了，是该解决一下了！！！

首先，在source文件夹中建立一个叫CNAME的文件（不要后缀），打开文件输入你的域名，比如我的是abellmz.cn，这就可以了，不需要https://

hexo clean、hexo g、hexo d 一波操作之后，去配置域名解析

然后，静等10min，然后打开abellmz.cn。恩！访问量未清零！
